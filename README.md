Rose is a next-generation Roblox monorepository toolchain handling:

- Workspace members and dependencies
- Alias-to-Roblox require resolution and injectable globals
- Novel sourcemap-based change watching for IDLs and code compilation

Rose will have:

- Documentation (soon!)
- Incremental compilation
- Deployment of games

Rose is battle-tested with:

- [Welcome To Hell by Team Fireworks](https://welcomestohell.com/)

## Installation

```sh
rokit add teamfireworks/rose
```
